## 自己的做法

### 算法思想

对于每一个`x/`记录，都代表进入了当前文件夹下的一个子文件夹。`../`代表返回上一个文件夹。

所以使用栈来存所有有效的记录，有效的记录是指`x/`型记录。

首先遍历logs数组，如果当前遍历字符串时`x/`，则入栈；如果是`../`则弹出栈顶元素；如果是`./`，则什么都不做。

最后栈中元素个数就是返回主文件夹所需的最小步数。



### 算法实现

```c++
class Solution {
public:
    int minOperations(vector<string>& logs) {
        stack<string> s;
        for(string str : logs) {
            if(str == "../") {
                if(!s.empty()) {
                    s.pop();
                }
            } else if(str == "./") {

            } else {
                s.push(str);
            }
        }
        return s.size();
    }
};
```
